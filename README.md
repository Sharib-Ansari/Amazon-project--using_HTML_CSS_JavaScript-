# 🛒 Amazon Clone

A modern and responsive Amazon-inspired e-commerce website built using HTML, CSS, and JavaScript. This project recreates the look and feel of Amazon's user interface while focusing on frontend development concepts, responsive design, and interactive user experience.

---

## 📖 Overview

The Amazon Clone project was developed to strengthen my understanding of frontend web development by recreating one of the world's most popular e-commerce platforms.

The project demonstrates how HTML, CSS, and JavaScript can be combined to build visually appealing and interactive web applications.

---

## ✨ Features

* Responsive navigation bar
* Product showcase section
* Search bar interface
* Hero banner section
* Product cards and categories
* Interactive user interface
* Responsive design for different screen sizes
* Modern and clean layout
